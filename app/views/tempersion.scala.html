How does the conversion work?

Celsius Fahrenheit Kelvin Rankine
Celsius NA F = C x (9/5) + 32 K = C + 273.15 R = (C + 273.15)*(9/5)
Fahrenheit C = (F-32) x (5/9) NA K = (F + 459.67) x (5/9) R = F + 459.67
Kelvin C = K - 273.15 F = K x (9/5) - 459.67 NA R = K x (9/5)
Rankine C = (R - 491.67) x (5/9) F = R - 459.67 K = R x (5/9) NA

What is "Tempersion"?

Tempersion is an online temperature conversion application by HWHelp. The table above shows the formulas used to calculate the conversions.
